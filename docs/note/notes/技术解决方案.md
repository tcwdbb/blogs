## 移动端特殊样式
 - 点击高亮需要清除 设置transparent
    
    ``` css 
    -webkti-tap-highlight-color: transparent
    ```
- 在移动端浏览器默认的外观在ios上加上这个属性才能给按钮和输入框自定义样式

   ``` css 
     -webkit-appearance: none
   ```
- 禁用长按页面时的弹出菜单

    ``` css
      img,a { -webkit-touth-callout: none }
    ```
